# Portfolio - Abderrahmen GHROUBI

Portfolio web responsive présentant mon parcours.

## Fonctionnalités

- **Design responsive** mobile/desktop
- **Mode sombre/clair** avec thèmes personnalisables

- **Technologies** : HTML5, CSS3, JavaScript vanilla
