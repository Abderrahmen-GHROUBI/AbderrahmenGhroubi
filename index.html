Tunisian Date Varieties Classification

Classification des Variétés de Dattes Tunisiennes

Python
TensorFlow
CNN
Computer Vision
Deep Learning

This project develops an image classification model to identify 11 varieties of Tunisian dates using the "Deglet Nour Date Fruit Dataset":

  • Dataset: Contains 11 date varieties with 300-473 images per class, including varieties like Alig, Bessra, Deglet Nour (multiple types), Kenta, and Kintichi.
  • Techniques used:
    • Image preprocessing (resizing, normalization, contrast enhancement)
    • Data augmentation (rotation, shearing, zoom, flipping)
    • Convolutional Neural Networks (CNN)
    • Stratified cross-validation
    • Advanced evaluation metrics
  • Results: The model achieves a test accuracy of approximately 58%, with performance varying by date variety. Detailed results include confusion matrices, classification reports, and learning curves.

Ce projet développe un modèle de classification d'images pour identifier 11 variétés de dattes tunisiennes en utilisant le jeu de données "Deglet Nour Date Fruit Dataset" :

  • Jeu de données : Contient 11 variétés de dattes avec 300-473 images par classe, incluant des variétés comme Alig, Bessra, Deglet Nour (plusieurs types), Kenta, et Kintichi.
  • Techniques utilisées :
    • Prétraitement d'images (redimensionnement, normalisation, amélioration du contraste)
    • Augmentation de données (rotation, cisaillement, zoom, retournement)
    • Réseaux de neurones convolutifs (CNN)
    • Validation croisée stratifiée
    • Métriques d'évaluation avancées
  • Résultats : Le modèle atteint une précision de test d'environ 58%, avec des performances variables selon les variétés de dattes. Les résultats détaillés comprennent des matrices de confusion, des rapports de classification et des courbes d'apprentissage.
